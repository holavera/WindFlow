¿Qué es WindFlow?

WindFlow es tu asistente digital para aumentar la productividad. Administra tus tareas, tus notas y accede a modos de enfoque con técnicas Pomodoro. Diseñado para adaptarse a ti, desde usuarios casuales hasta profesionales suscritos o administradores.

Comparativa de Roles en WindFlow

Invitado

  • Notas (no guardadas)
  • 3 Modos Pomodoro

Usuario Gratuito

  • Notas ilimitadas
  • Tareas ilimitadas
  • 3 Modos Pomodoro
  • Winds ilimitados

Usuario Suscrito

  • Notas y Tareas ilimitadas
  • 3 Modos Pomodoro
  • Winds ilimitados
  • Ver Estadísticas
  • Ver Historial Pomodoro
  • Exportar PDF

Administrador

  • Gestión de usuarios y grupos
  • Creación de winds para usuarios
  • Estadísticas por usuario
  • Exportar PDF
